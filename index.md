# Amino Acid Curriculum

Welcome to the Amino Acid Curriculum documentation.

## Overview
This project provides an interactive and comprehensive educational tool for biochemistry students and enthusiasts. It features 2D and 3D visualizations of all 21 human amino acids and their corresponding mRNA codons.
